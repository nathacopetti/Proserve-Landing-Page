Meu primeiro projeto mobile-first: PROSERVE.

O PROSERVE é um projeto landing-page de uma página de academia.

No processo de aprendizagem, um dos projetos sugeridos no roadmap feito pela IA, foi de desenvolver um projeto Mobile-First.

Neste projeto, comecei o desenvolvimento visando a visibilidade em dispositivos móveis, e depois portei isso para tablets e posteriormente ao desktop.

Apesar de ainda se tratar de uma página estática, estou feliz com o processo de aprendizagem. Não é uma página perfeita, mas definitivamente é um produto do meu esforço diário de codar um pouco por dia.

Utilizei media-queries, flexbox, e entendi um pouco sobre o uso de width e height 100% - mais um passo no entendimento da parentalidade em desenvolvimento web. 

Espero olhar esse projeto no futuro e perceber o quanto mudei pra melhor! Bora!
